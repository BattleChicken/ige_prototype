IgeMousePanComponent CLASS

Declared in IgeMousePanComponent.js

Description
When added to a viewport, automatically adds mouse panning capabilities to the viewport's camera.
Extends
IgeEventingClass

Methods

init({IgeObject} entity, {Object} options)

  • {IgeObject}entity The object that the component is added to.
  • {Object}options The options object that was passed to the component during the call to addComponent.

startThreshold({} val)

Gets / sets the number of pixels after a mouse down that the mouse must move in order to activate a pan operation. Defaults to 5.
  • {}val
Returns {*}

limit({IgeRect} rect)

Gets / sets the rectangle that the pan operation will be limited to using an IgeRect instance.
  • {IgeRect}rect
Returns {*}

enabled({Boolean} val)

Gets / sets the enabled flag. If set to true, pan operations will be processed. If false, no panning will occur.
  • {Boolean}val
Returns {*}

_mouseDown({} event)

Handles the mouseDown event. Records the starting position of the camera pan and the current camera translation.
  • {}event

_mouseMove({} event)

Handles the mouse move event. Translates the camera as the mouse moves across the screen.
  • {}event

_mouseUp({} event)

Handles the mouse up event. Finishes the camera translate and removes the starting pan data.
  • {}event
