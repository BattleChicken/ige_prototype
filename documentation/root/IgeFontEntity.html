IgeFontEntity CLASS

Declared in IgeFontEntity.js

Description
Creates a new font entity. A font entity will use a font sheet (IgeFontSheet) and render text based on that font sheet's glyphs.
Extends
IgeUiEntity

Methods

colorOverlay(String val)

Gets / sets the string hex or rgba value of the colour to use as an overlay when rending this entity's texture.
  • {String}val The colour value as hex e.g. '#ff0000' or as rgba e.g. 'rbga(255, 0, 0, 0.5)'. To remove an overlay colour simply passed an empty string.
Returns {*} "this" when arguments are passed to allow method chaining or the current value if no arguments are specified.

clearCache()

Clears the texture cache for this entity's text string.

nativeFont(String val)

When using native font rendering (canvasContext.fillText()) this sets the font and size as per the canvasContext.font string specification.
  • {String}val The font style string.
Returns {*} "this" when arguments are passed to allow method chaining or the current value if no arguments are specified.

_stringify()

Returns a string containing a code fragment that when evaluated will reproduce this object's properties via chained commands. This method will only check for properties that are directly related to this class. Other properties are handled by their own class method.
Returns {String}
