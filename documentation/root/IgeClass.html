IgeClass CLASS

Declared in IgeClass.js

Description
The base class system.

Methods

log(String text, String type, Object obj)

Provides logging capabilities to all IgeClass instances.
  • {String}text The text to log.
  • {String}type The type of log to output, can be 'log', 'info', 'warning' or 'error'.
  • {Object}obj An optional object that will be output before the log text is output.

classId()

Returns the class id. Primarily used to help identify what class an instance was instantiated with and is also output during the ige.scenegraph() method's console logging to show what class an object belongs to.

addComponent(IgeClass component, Object options)

Creates a new instance of the component argument passing the options argument to the component as it is initialised. The new component instance is then added to "this" via a property name that is defined in the component class as "componentId".
  • {IgeClass}component The class definition of the component.
  • {Object}options An options parameter to pass to the component on init.

removeComponent(String componentId)

Removes a component by it's id.
  • {String}componentId The id of the component to remove.

implement(Function classObj, Boolean overwrite)

Copies all properties and methods from the classObj object to "this". If the overwrite flag is not set or set to false, only properties and methods that don't already exists in "this" will be copied. If overwrite is true, they will be copied regardless.
  • {Function}classObj
  • {Boolean}overwrite

data(String key, * value)

Gets / sets a key / value pair in the object's data object. Useful for storing arbitrary game data in the object.
  • {String}key The key under which the data resides.
  • {*}value The data to set under the specified key.
